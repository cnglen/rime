#+TITLE: rime

* 配置

- build目录为部署后自动生成, 可删除，重新部署会自动生成。


- rime的前端
  - macOS: Squirrel(鼠须管)
  - windows: Weasel(小狼毫)
  - linux: ibus-rime
    
针对个人习惯，进行了一点微调。



default.custom.yaml 中的 schema_list 字段


配置文件:
- default.custom.yaml: 全局设定
- <方案標識>.custom.yaml：預設輸入方案副本
- 〔發行版設定〕 ibus_rime.default.yaml

* 安装

#+begin_src
cd .config/ibus/
git clone git@github.com:cnglen/rime.git
cd rime && make
#+end_src


https://extensions.gnome.org/extension/4112/customize-ibus/ 安装，支持ibus的配置（字体大小等）


** 如何更新配置



** 配置文件:

目录:
- linux: ~/.config/ibus/rime
- mac:  ~/Library/Rime
- windows: %APPDATA%\Rime 

文件:

- default.custom.yaml: 全局设定
- <方案標識>.custom.yaml：預設輸入方案副本
- 發行版設定:
  - linux: ibus_rime.default.yaml
  - mac: squirrel.custom.yaml
